🔌 GitHub সংযোগ

যে repo তে config.json রাখবেন, তার তথ্য দিন। Token শুধু এই ব্রাউজার সেশনে থাকে, save হয় না।

প্রথমবার হলে repo তে খালি ফাইল {"access_control":{"enabled":true,"users":[]},"redirect":{"enabled":false,"url":"","interval_seconds":300}} বানিয়ে commit করে রাখুন — তারপর Load চাপুন।

🔐 Access Control

শুধু নিচের তালিকার Device ID + License Key মিললেই অ্যাপ ব্যবহার করতে পারবে।

Access Control চালু রাখুন
বন্ধ করলে সবাই অ্যাপ ব্যবহার করতে পারবে (whitelist ignore হবে)
Device ID License Key নাম / নোট Active
